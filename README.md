# SocialSphere 🚀

A full-stack social media platform built with the MERN Stack.

## Features

* User Registration & Login
* JWT Authentication
* Protected Routes
* Create Posts
* Edit Posts
* Delete Posts
* Like / Unlike Posts
* Comment System
* MongoDB Database
* Responsive UI

## Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios
* CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
